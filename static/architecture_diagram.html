Structural Design of the Model — Architectural Layers

Layer 1 — Data Input

Structured Data

  • Historical stock prices
  • Trading volumes
  • Technical indicators (MA, RSI, momentum)
  • Economic indicators (GDP, interest rates)

Unstructured Data

  • Financial news articles
  • Earnings announcements
  • Social media content
  • Analyst commentary
Preprocessing & Transformation
Layer 2 — Preprocessing & Feature Engineering (KDD / CRISP-DM)

Structured Pipeline

  • Data cleaning & normalisation
  • Feature selection & engineering
  • Time-series segmentation
  • Trend labelling

NLP Pipeline

  • Text cleaning & tokenisation
  • Sentiment classification
  • Event extraction
  • Sentiment score generation

Data Fusion

  • Align structured & sentiment data by timestamp
  • Merge into unified feature set
  • Prepare training & test datasets
Modeling
Layer 3 — Modeling

Traditional Signal Scorer

  • RSI, MA7 / MA30 composite scoring
  • Linear regression forecast
  • Buy / Hold / Sell direction
  • Input: fused structured features

Cox PH Survival Model

  • Time-to-event modelling
  • Hazard & survival function estimation
  • Time-to-reversal prediction
  • Input: detected trend + market features

LLM Output Module

  • Receives Cox + signal outputs
  • Generates natural language insight
  • Contextual stock analysis chat
  • Input: statistical model results
